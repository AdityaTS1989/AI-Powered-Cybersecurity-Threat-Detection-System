# 🔐 AI-Powered Cybersecurity Threat Detection

## 🚀 Overview

A machine learning system that detects cyber threats from network traffic data.
It identifies suspicious behavior using a Random Forest model.

---

## 🎯 Problem Statement

Traditional rule-based systems fail to detect new cyber threats.
This project uses AI to detect anomalies in network traffic.

---

## ⚙️ Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Matplotlib

---

## 🤖 Model

* Random Forest Classifier
* Trained on labeled cybersecurity dataset
